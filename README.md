### 蜗牛农场
> **可以在线下播种的线上农场**


## 开发环境
  - [Android 3.2](https://developer.android.google.cn/)
  - [NOX模拟器](https://www.yeshen.com/)
  - Android 5.0

## 技术实现
+	使用Bmob实现数据库存储与配置
+ 使用Fragment与ViewPage实现页面切换
+	实现了夜间模式与正常模式的切换
+	使用WebView外接果蔬百科与天气预报
+	使用快递鸟API实现物流信息的查询
+	实现自动登录与清除登录痕迹
+	利用CardView+RecycleView构建列表
